# European Sales Dashboard

## Overview
Interactive Tableau dashboard providing sales performance analysis across European countries, regions, products, and customer segments.

## Dashboard Features

- Executive KPI Cards
- Sales Trend Analysis
- Country Performance Analysis
- Product Category Performance
- Customer Insights
- Interactive Filters

## Tools Used

- Tableau
- SQL
- Data Analysis
- KPI Reporting

## Business Value

Provides leadership and business stakeholders with visibility into sales performance, profitability trends, customer behavior, and regional growth opportunities.

## Skills Demonstrated

- Data Visualization
- KPI Analytics
