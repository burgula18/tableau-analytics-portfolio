# Adidas USA Sales Dashboard

## Overview
This Tableau dashboard provides executive-level insights into Adidas sales performance across the United States. The dashboard enables business users to monitor sales trends, regional performance, customer behavior, and key performance indicators.

## Key Metrics
- Total Sales
- Profit Analysis
- Regional Performance
- State-wise Sales Distribution
- Product Performance
- Customer Insights

## Tools Used

- Business Intelligence

## Skills Demonstrated

Provides stakeholders with actionable insights to support sales monitoring, performance management, and strategic decision-making.
