# Tableau Analytics Portfolio

## Overview

This repository contains a collection of Tableau dashboards developed to demonstrate data visualization, KPI reporting, executive analytics, and business intelligence capabilities.

The dashboards focus on transforming business data into actionable insights through interactive visualizations, performance tracking, trend analysis, and executive reporting.

---

## Dashboards Included

### 1. Adidas USA Sales Dashboard
Business sales performance dashboard analyzing:
- Sales trends
- Regional performance
- Product category analysis
- Executive KPI reporting

### 2. European Sales Dashboard
Interactive dashboard focused on:
- Revenue analysis
- Profitability metrics
- Geographic performance
- Customer insights

### 3. Impact of COVID on Air Travel
Analytical dashboard highlighting:
- Passenger traffic trends
- Airline performance
- COVID impact analysis
- Travel industry insights
